My name is Jennifer and I am a junior frontend developer, with a background as an experienced graphic designer and magazine art director. The past six years my main assignment has been as the art director for the magazine Scandinavian Retro (Egmont). I have freelanced at magazines and advertising agencies as art director/graphic designer, photographer and web editor for the last 15 years. I have worked at Bonnier, Forma Publishing as well as Egmont. In 2012 I won oneshot of the year at the 2012 Swedish Magazine Awards.

Currently, I'm completing the Technigo Frontend Bootcamp - adding React, React Native, Javascript, CSS and web design to my palette. And of course I'm really eager to put my new skills to work. I'm a team player, I'm prestigeless, and always going for the best solution of a problem.

I live on Lilla Essingen in Stockholm with my husband, two kids and a dog. I'm also pretty good at solving Rubiks cube (for being a mom), I participated in Rubiks cube world championship 2019.

POPULAR MOVIES

POPULAR MOVIES SITE. I built a multi-page React application that shows the most popular movies in the US. I started by sketching how many pages I wanted to display and which components to use. Then I used postman to check out the api and how the information was set up. I worked in React using useState, useEffects, Route, Switch and BrowserRouter to enable the functionality in this project. If I had more time, I would have made a nav bar and bring in more information from different endpoints in the api. >>

WEATHER APP

WEATHER APP. This project was to build a weather web app that shows today's weather and temperature, and a 5-day forecast using a weather API. I started with checking out the API and tried to figure out how it worked and what kind of information I could fetch from it. I made a few conditionals to change the color of a section depending on the temperature. I did this with classList.toggle. I also used conditionals to change the color of the headline depending on the weather description. I did that by changing style.color in the js-file. I used forEach to iterate through the forecast API to get the days and temperature for each day. If I had more time, I would have done a dropdown menu so I would be able to choose the weather in a few different citys. I would also have done it responsive, as it's made now it works best for the phone. >>
